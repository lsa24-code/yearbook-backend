# yearbook-backend
API REST do Yearbook Digital — IFNMG Salinas 2026
Título: Site de engenharia civil da Contrutora Alves

Descrição:O projeto "Construtora Alves" é um site desenvolvido para apresentar os serviços e projetos de uma empresa de engenharia e construção civil. O objetivo da plataforma é facilitar a divulgação da empresa, apresentar seu portfólio e permitir o contato entre clientes e a equipe da construtora.

Principais Funcionalidades:
- Página inicial (Home) com apresentação da empresa;
- Página "Construtora Alves" contando a história da empresa;
- Página "Nossos Serviços" com os serviços oferecidos;
- Portfólio com projetos finalizados e em andamento;
- Página "Trabalhe Conosco" para envio de interesse em participar da equipe;
- Área administrativa do sistema.
